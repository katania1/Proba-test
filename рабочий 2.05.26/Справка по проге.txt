

# 📖 Руководство пользователя VibeCoder v1.15 (Pro)

Добро пожаловать в **VibeCoder** — вашу персональную среду для профессионального "вайбкодинга". Эта программа работает как умный мост между вашим локальным проектом и веб-версией нейросети Gemini 3.1 Pro. Она защищает ваш код, следит за контекстом и автоматизирует рутину.

---

## 1. Интерфейс и Базовые настройки
Рабочая область VibeCoder разделена на логические зоны. Вы можете изменять их размер, перетягивая разделители.

*   **Независимый Зум:** Вы можете зажать `Ctrl` и покрутить колесико мыши, чтобы увеличить или уменьшить шрифт в трех зонах: в левом дереве файлов, в логах чата и в самом редакторе кода. Программа запомнит ваш выбор для следующих сессий.
*   **Счетчик токенов:** В правом нижнем углу окна в реальном времени отображается примерное количество токенов в вашем текущем запросе (рассчитывается математически).
*   **Выбор проекта:** Нажмите кнопку `📂` на панели над деревом файлов, чтобы переключаться между разными папками-проектами. Программа запомнит последнюю открытую папку.

---

## 2. Общение с ИИ и Генерация кода
Основная концепция программы — вы ставите задачи на человеческом языке, а ИИ пишет или изменяет код.

*   **Отправка запросов:** Введите задачу в нижнее текстовое поле и нажмите `Ctrl + Enter` (обычный `Enter` делает перенос строки).
*   **Drag & Drop файлов:** Чтобы ИИ понял, с каким файлом ему работать, зажмите файл в левом дереве левой кнопкой мыши и перетащите в текстовое поле чата. Появится тег упоминания (например, `@[core/main.py]`).
*   **Прикрепление кода:** Если вам нужно, чтобы ИИ прочитал текущий код файла перед тем как писать ответ, кликните по тегу `@[имя_файла]` в чате **правой кнопкой мыши (ПКМ)**. Тег станет зеленым — это значит, что при отправке запроса код этого файла будет вложен внутрь.
*   **Авто-контекст (Tool Calling):** ИИ сам умеет запрашивать чтение нужных ему файлов. Если вы не прикрепили код, но ИИ он нужен, программа выдаст всплывающее окно с вопросом: *"ИИ просит прочитать эти файлы, разрешить?"*.
*   **Умный патчинг (Smart Diff):** ИИ не присылает файлы целиком. Он использует точечные команды "найти и заменить". При получении ответа откроется окно **Diff Viewer**, где красным будет показан старый код, а зеленым — новый. Нажмите **"✅ Утвердить"**, чтобы применить изменения.
*   **Авто-Валидатор:** Если ИИ ошибается (пишет сломанный код или ленивые заглушки "остальной код здесь"), программа сама это заметит, заблокирует ответ и отправит ИИ гневное требование всё исправить.

## 3. Управление проектом (Левая панель)
Дерево файлов слева — это ваш командный пункт.

*   **Мини-панель создания:** Используйте иконки над деревом для создания новых файлов `📄` или папок `📁`, а также для переименования `✏️` и безопасного удаления `🗑️`.
*   **Smart .gitignore:** Если в корне проекта есть файл `.gitignore`, программа автоматически прочитает его. Все игнорируемые файлы и папки (например, кэш или виртуальные среды) станут темно-серыми. 
*   **Быстрый игнор:** Кликните ПКМ по любой папке или файлу в дереве и выберите **"🚫 Добавить в .gitignore"**, чтобы скрыть их от Git.
*   **Вкладки редактора:** Кликнув на файл, вы откроете его в редакторе. Сверху появятся вкладки (Tabs) для удобного переключения между открытыми скриптами. При закрытии вкладки файл не удаляется с диска.

## 4. Двойная система бэкапов (Защита кода)
В VibeCoder внедрена двухуровневая система сохранения, чтобы вы никогда не потеряли свой труд.

### Уровень 1: Локальная "Машина Времени" (Для минутных ошибок)
Срабатывает автоматически при каждом ручном сохранении (`Ctrl+S`) или при утверждении кода от ИИ.
*   Хранит ровно **10 последних версий** каждого файла в скрытой папке `.vibe_backups/`.
*   **Как использовать:** Кликните ПКМ по сломанному файлу в левом дереве проекта -> выберите **"🕒 Машина Времени"**. Откроется окно, где можно посмотреть отличия старой версии от текущей и восстановить файл в один клик.

### Уровень 2: Профессиональный Git и GitHub (Для вечного хранения)
Срабатывает только вручную по нажатию на кнопку **`📦 Git`** в нижней панели управления. Цвет кнопки меняется на оранжевый, если в проекте есть несохраненные изменения.
В открывшемся окне вам доступны три вкладки:
*   **Точечные Коммиты (Staging):** Вы можете выбрать галочками только те файлы, которые хотите сохранить в историю. 
*   **AI-Коммиты:** Нажмите "✨ Сгенерировать ИИ-описание", и нейросеть сама прочитает изменения и напишет профессиональный лог того, что вы сделали. Нажмите "✅ Сделать Commit" для фиксации на диске.
*   **Синхронизация с GitHub (Push/Pull):** Вставьте ссылку на ваш репозиторий. Используйте кнопку **"☁️ Push"** для отправки коммитов в облако.
*   **Хирургическое извлечение (Surgical Pull):** Если вы хотите скачать с GitHub не весь проект, а только один сломанный файл, поставьте галочку "Показать ВСЕ файлы", выделите нужный файл и нажмите фиолетовую кнопку **"🎯 Скачать выбранные"**.
*   **Облачная Git-История:** Кликните ПКМ по любому файлу прямо в окне Git и выберите "🕒 Посмотреть Git-историю". Вы увидите всю историю коммитов этого файла, сможете посмотреть код прошлых лет и сравнить Diff-патчи.

## 5. Браузерная интеграция и Вкладки
Для работы VibeCoder необходимо расширение Chrome.

*   **VibeBadge (Плашка):** На странице Gemini появится плашка (Heartbeat). Она показывает статус работы. Вы можете зажать иконку `⠿` левой кнопкой мыши и перетащить плашку в любое удобное место экрана.
*   **Имена вкладок:** Кликните по названию вкладки на плашке прямо в браузере, чтобы переименовать её (например, "Фронтенд", "Бэкенд").
*   **Мульти-маршрутизация:** В статус-баре IDE (внизу слева) есть выпадающий список. Выберите в нем нужную вкладку браузера, и VibeCoder будет отправлять задачи именно в нее. Это позволяет вести параллельные диалоги с ИИ для разных частей проекта.
*   **Кнопка "🔄 Эстафета":** Если текущий чат в браузере стал слишком длинным и ИИ начал "глючить", нажмите эту кнопку на нижней панели IDE. Программа сама запакует весь контекст проекта, откроет новую чистую вкладку в браузере и передаст туда всю эстафету.

---

## 6. Шпаргалка Горячих Клавиш
*   `Ctrl + Enter` — Отправить задачу ИИ.
*   `Ctrl + S` — Ручное сохранение файла + создание бэкапа в Машине Времени.
*   `Ctrl + Колесико мыши` — Зум интерфейса.
*   `Shift + Клик` (в дереве файлов) — Выделение нескольких файлов подряд.
*   `Ctrl + Клик` (в дереве файлов) — Точечное выделение нескольких файлов.
*   `ПКМ по файлу` — Вызов контекстного меню (Машина времени, gitignore).
*   `ПКМ по тегу в чате` — Включение/отключение прикрепления кода.
